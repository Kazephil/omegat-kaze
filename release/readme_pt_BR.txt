O que é OmegaT?
===============

O OmegaT é uma ferramenta de tradução assistida por computador gratuita e de código aberto com correspondência parcial, memória de tradução, busca de palavras-chave, glossários e reutilização de tradução em atualização de projetos traduzidos.

Pré-requisitos para a instalação do OmegaT
==========================================

O OmegaT requer o Java Runtime Environment (JRE) versão 1.8 ou maior para ser instalado em seu sistema.

Recomendamos usar os pacotes do OmegaT que incluem o JRE para poupá-lo do trabalho de instalá-lo.



Instalação do OmegaT (Windows)
==============================

Inicializar o programa de instalação



Instalação do OmegaT (Mac)
==========================

Descompactar o arquivo .zip do OmegaT para obter uma pasta que contém um arquivo de documentação e a ferramenta OmegaT. Mover a pasta para um local apropriado tal como a pasta Ferramentas.



Instalação do OmegaT (Linux)
============================

Coloque o arquivo em qualquer pasta adequada e descompacte-o. O OmegaT está, então, pronto para ser inicializado.

No entanto, você pode ter uma instalação mais fácil e mais amigável usando o script de instalação (linux-install.sh). Para usar esse script, abra um janela de terminal (console), mude a pasta para a pasta que contém OmegaT. jar e o script linux-install.sh, e execute o script com ./linux-install.sh.



Instalação do OmegaT (Solaris, FreeBSD, etc.)
=============================================

Coloque o arquivo em qualquer pasta adequada e descompacte-o. O OmegaT está, então, pronto para ser inicializado.



Iniciando o OmegaT (Windows)
==========================

Se, durante a instalação, você criou um atalho no desktop, dê duplo clique nesse atalho.

O programa de instalação pode criar atalhos para você no menu iniciar, na área de trabalho e na barra de início rápido. Você também arrastar o arquivo OmegaT.exe manualmente para o menu iniciar, para a área de trabalho ou para a barra de início rápido para criar um vínculo.

Se você pode ver o arquivo OmegaT mas não o OmegaT.exe no seu gerenciador de arquivos (Windows Explorer), mude as configurações para que as extensões de arquivo sejam mostradas.



Iniciando o OmegaT (Mac)
========================

Duplo clique na ferramenta OmegaT.

Você pode arrastar o aplicativo OmegaT para o dock ou para a barra de ferramentas de uma janela do Finder para poder iniciá-lo em qualquer local. Você também pode iniciar o OmegaT a partir do campo pesquisa Spotlight.



Iniciando o OmegaT (Linux)
==========================

Se você usou o script linux-install.sh, você deve conseguir iniciar o OmegaT com:

    Alt+F2

e então:

    omegat

Para uma maneira mais fácil de abrir o OmegaT, você pode usar o script Kaptain fornecido
(omegat.kaptn). Instale o Kaptain para poder usar o script. Então,
execute o script de inicialização do Kaptain conforme os seguintes passos:

    Alt+F2

e então:

    omegat.kaptn



Iniciando o OmegaT a partir da linha de comando (em qualquer sistema operacional)
=================================================================================

O comando para iniciar o OmegaT é:

    cd <pasta que contém o arquivo OmegaT.jar>

    <nome e caminho do arquivo executável java> -jar OmegaT.jar

(O arquivo Java executável é o arquivo java no Linux e o java.exe no Windows.  Se o Java estiver instalado a nível de sistema e estiver no caminho de comando, o caminho completo não precisará ser inserido.)
